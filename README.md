# open-banking-account-aggregation
Open Banking Ecosystem - India Chapter - RBI Directive | Account Aggregation APIs using Spring Boot

Entities:
Central Repository - Registry of all entities in an Open Banking Ecosystem
FIP - Financial Information Provider e.g. Any Bank
Account Aggregator - API infrastructure PaaS that acts as an intermediary between an FIP and end user.

This project is a Proof of Concept(POC) of the Account Discovery feature in an Account Aggregator Application.

Guidlines and Specifications defined by ReBIT - The IT wing of RBI in the following link:
https://api.rebit.org.in/
